# Phase 4 최종 보고서: TTS 모델 통합 비교 및 서비스 권장안

본 보고서는 MeloTTS, Supertonic2, Qwen3-TTS 세 모델에 대한 기술적 벤치마크(Codex)와 청취 품질 평가(Gemini)를 통합하여, 사용자 및 서비스 관점에서의 최종 의사결정 가이드를 제공합니다.

---

## 1. 3사 모델 종합 비교표

| 평가 항목 | Supertonic2 | MeloTTS | Qwen3-TTS | 최우수 모델 |
|---|:---:|:---:|:---:|:---:|
| **한국어 자연성 (MOS)** | 4.2 | 3.8 | **4.5** | **Qwen3-TTS** |
| **명료성 (Intelligibility)** | **4.5** | 4.0 | **4.5** | **Supertonic / Qwen3** |
| **스타일 제어 유연성** | 낮음 (프리셋 10종) | 매우 낮음 (속도만) | **매우 높음 (자연어 지시)** | **Qwen3-TTS** |
| **특수 기능 (클로닝 등)** | 없음 | 없음 | **있음 (Zero-shot)** | **Qwen3-TTS** |
| **생성 속도 (Avg RTF)** | **0.024 (압도적)** | 0.223 | 0.838 | **Supertonic2** |
| **운영 비용 (HW)** | **낮음 (CPU)** | 중간 (CPU) | 높음 (GPU 필수) | **Supertonic2** |
| **설치 및 배포 난이도** | **낮음** | 중간 | 높음 | **Supertonic2** |

## 2. 유스케이스별 최종 추천 모델

### **① 저사양 CPU 챗봇 / 실시간 안내 시스템**
- **추천 모델**: **Supertonic2**
- **사유**: RTF 0.024의 압도적인 속도로 저사양 CPU에서도 지연 없는 응답이 가능합니다. 10종의 한국어 프리셋이 이미 훌륭한 품질을 갖추고 있어 별도의 튜닝 없이 즉시 투입 가능합니다.

### **② 고속 배치 오디오 생성 (오디오북, 대량 컨텐츠)**
- **추천 모델**: **Supertonic2**
- **사유**: 초당 약 143자를 처리하는 높은 처리량(Throughput)으로 대량의 텍스트를 가장 짧은 시간과 낮은 비용으로 오디오화할 수 있습니다.

### **③ 글로벌 액센트 지원 서비스 (영어권 타겟)**
- **추천 모델**: **MeloTTS**
- **사유**: 미국(US), 영국(BR), 인도(India), 호주(AU) 등 다양한 영어 액센트를 기본 지원하며, 각 액센트의 특징을 매우 잘 살립니다. 글로벌 확장이 필요한 서비스에 최적입니다.

### **④ 프리미엄 감정 표현 서비스 (스토리텔링, 게임)**
- **추천 모델**: **Qwen3-TTS (CustomVoice)**
- **사유**: 자연어 지시(Instruct)를 통해 기쁨, 슬픔, 분노 등 세밀한 감정 연출이 가능합니다. 특히 구연동화 스타일(MOS 4.7)은 실제 성우 수준의 표현력을 보여줍니다.

### **⑤ 사용자 음성 클로닝 및 개인화 앱**
- **추천 모델**: **Qwen3-TTS (Base / VoiceDesign)**
- **사유**: 3초 내외의 짧은 샘플만으로 화자의 정체성을 복제하는 Zero-shot 클로닝 기능과, 텍스트 묘사만으로 목소리를 설계하는 VoiceDesign 기능은 Qwen3-TTS만의 독보적인 강점입니다.

## 3. 모델별 "주의사항 (쓰지 말아야 할 상황)"

| 모델 | 사용을 권장하지 않는 상황 |
|---|---|
| **Supertonic2** | 사용자 맞춤형 목소리(클로닝)가 필수적이거나, 자연어 지시를 통한 정밀한 감정 제어가 필요한 경우. |
| **MeloTTS** | 높은 수준의 한국어 자연성이 요구되거나, GPU 서버 없이 실시간 고속 응답이 핵심인 서비스. |
| **Qwen3-TTS** | 예산이 극히 제한적인 프로젝트, GPU 가용성이 없는 환경, 혹은 단순 텍스트 전달이 목적인 저사양 시스템. |

## 4. 최종 결론 (Gemini's Take)

본 탐색 작업 결과, **"만능 모델은 없으며 목적에 따른 명확한 분리가 필요하다"**는 결론에 도달했습니다.

- **성능과 비용 효율성**이 최우선이라면 **Supertonic2**를 기반으로 시스템을 구축하십시오.
- **사용자 경험의 깊이와 개인화**가 최우선이라면 **Qwen3-TTS**를 선택하되, GPU 인프라 비용을 고려하십시오.
- **글로벌 시장 대응**이 목적이라면 **MeloTTS**의 영어 액센트 기능을 혼용하는 전략이 유효합니다.

운영 측면에서는 **Supertonic2를 메인 엔진**으로 사용하고, **Qwen3-TTS를 프리미엄 기능(클로닝, 감정 연출)을 위한 서브 엔진**으로 배치하는 하이브리드 아키텍처를 권장합니다.
